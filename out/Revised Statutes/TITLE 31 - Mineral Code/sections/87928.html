PART 4.  APPLICATION OF CODE

§214.  Applicability of Code to existing rights

The provisions of this Code shall apply to all mineral rights, including those existing on the effective date hereof; but no provision may be applied to divest already vested rights or to impair the obligation of contracts.  

Acts 1974, No. 50, §214, eff. Jan. 1, 1975.
