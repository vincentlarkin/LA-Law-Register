§1122.104.  Description of units

After the declaration is properly filed for registry, a description of a unit which sets forth the name of the condominium, the place of recordation of the declaration, the parish in which the condominium is located, and the identifying number of the unit, is a sufficient legal description of that unit and its common element interest even if the common element interest is not described or referred to therein.  

Acts 1979, No. 682, §1.
