Art. 110.  References to code articles or statutory sections
Unless the context clearly indicates the contrary:
(1)  A reference in this Code to a Title, Chapter, or Article without further designation, means a Title, Chapter, or Article of this Code.
(2)  A reference in this Code to an Article of a Code or to a statutory section applies to subsequent amendments thereof.
Acts 1991, No. 235, §1, eff. Jan. 1, 1992.
